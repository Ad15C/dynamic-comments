# CV HTML CSS

## Description
CV en ligne développé en HTML5 et CSS3 dans le but de présenter mon profil de développeuse web de manière claire, structurée et responsive.

## Objectifs
- Structurer une page web avec HTML
- Mettre en forme un CV avec CSS
- Créer une interface lisible et professionnelle
- Travailler la mise en page responsive

## Technologies utilisées
- HTML5
- CSS3

## Fonctionnalités
- Présentation du profil
- Affichage des compétences
- Mise en avant des expériences et formations
- Mise en page responsive

## Installation et utilisation
Aucune installation n’est nécessaire.

```bash
git clone https://github.com/Ad15C/cv-html-css.git
```

## Structure du projet
* index.html : structure de la page
* style.css : mise en forme du CV
* assets/ : images, icônes ou autres ressources

## Aperçu
Ce projet présente un CV en ligne mettant en avant :
* les compétences
* les expériences professionnelles
* les formations

* Ajouter une version mobile plus poussée
* Améliorer l’accessibilité
